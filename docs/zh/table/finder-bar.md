# 查找器

-   由于虚拟表格无法被系统级的 Ctrl + F 检索到内容，因此需要通过自定义搜索逻辑来实现数据查找。
-   性能内置做了分批查找（注意数据越多查找越久）
-   查找成功会自动滚动道指定位置并设置背景色

## Config

| 参数                 | 说明                   | 类型    | 默认值             |
| -------------------- | ---------------------- | ------- | ------------------ |
| ENABLE_FINDER        | 启用查找器             | boolean | true               |
| FINDER_CELL_BG_COLOR | 当前查找结果格子背景色 | string  | `rgb(255,229,144)` |
