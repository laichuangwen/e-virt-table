## EVirtTableVue组件
 基于vue2和element-ui封装的组件
- 拓展编辑器(支持下拉、日期、时间、tree等)
- 拓展空数据
- 拓展覆盖层，支持dom图片等
- 支持不同格子有不同编辑器
::: warning 注意
EVirtTableVue 只是我个人封装的一个组件示例，你可以基于 e-virt-table 核心库，并参考我的示例，结合其他流行的 UI 库，实现更丰富、功能更强大的表格组件。
:::
